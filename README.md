# Lost and Found Item Finder

## Overview

Lost and Found Item Finder is a web application that helps users report, search for, and recover lost belongings. Users can post details about lost or found items, upload images, specify locations, and connect with others to facilitate the safe return of items.

## Features

* User Registration and Login
* Report Lost Items
* Report Found Items
* Upload Item Images
* Search and Filter Listings
* Item Categorization
* Location-Based Item Tracking
* Claim and Recovery Requests
* Responsive Design for Mobile and Desktop
* Secure User Authentication

## Installation

### Clone the Repository

```bash
git clone <repository-url>
cd lost-and-found-item-finder
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
MONGODB_URI=your_database_url
JWT_SECRET=your_jwt_secret
PORT=5000
```

### Run the Application

```bash
npm run dev
```

## Usage

1. Register or log in to your account.
2. Create a lost or found item report.
3. Add item details, images, and location information.
4. Search for matching items using filters.
5. Contact users and submit claims for item recovery.
